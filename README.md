# Mooteam

## Запуск

### Установка зависимостей:
`yarn`

### Запуск фронтенда
`yarn watch`

### Запуск фейкового бекенда
`yarn watch:db`

### Запуск storybook
`yarn watch:storybook`

## Кодстайл

### ESlint
`yarn watch:eslint`

### Stylelint
`yarn watch:stylelint`

### Prettier
`yarn watch:prettier`
